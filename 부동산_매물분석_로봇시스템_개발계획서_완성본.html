주요 유스케이스

UC01: 자동 평면도 생성
옴니휠 로봇이 매물을 순찰하며 정밀 평면도 자동 생성
UC02: 환경 품질 분석
온도, 습도, 공기질, 조도, 소음 등 종합 환경 측정
UC03: 매물 리포트 생성
측정 데이터 기반 전문적인 매물 분석 리포트 자동 작성
UC04: 웹 대시보드 제공
실시간 측정 진행상황 및 결과 웹 인터페이스 제공

시나리오 예시 - UC01

자동 평면도 생성
1. 부동산 중개사가 웹에서 새로운 매물 측정 요청
2. 현장에서 로봇 시작 버튼 클릭
3. 옴니휠 로봇이 모든 방을 자동 순찰
4. LiDAR로 벽, 문, 창문 위치 정밀 측정
5. 실시간으로 평면도가 웹에서 생성됨
6. 측정 완료 후 CAD 수준 평면도 자동 완성
7. 면적 계산 및 치수 표기 자동 추가
8. PDF 리포트 자동 생성 및 이메일 발송

• 계단 이동 불가 (엘리베이터 필요)
• WiFi 연결 필수 환경
